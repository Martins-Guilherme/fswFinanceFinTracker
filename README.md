# FinTrack

Controle suas finanças com praticidade e visualização inteligente.

---

![Dashboard FinTrack](/public/image.png)

## Sobre o projeto

O **FinTrack** é um aplicativo web construído com React para gerenciar saldo, ganhos, gastos e investimentos, permitindo o registro, filtro e análise de transações financeiras em tempo real. O dashboard oferece rápida visualização dos principais indicadores e histórico detalhado das movimentações.

---

## Funcionalidades

- Visualização de saldo, ganhos, gastos e investimentos.
- Cadastro, edição e remoção de transações por tipo e valor.
- Filtro por períodos e categorização de receitas e investimentos.
- Painel intuitivo com tema escuro nativo e estatísticas percentuais.
- Validação de formulários e feedback visual das ações.

---

## Tecnologias Utilizadas

- **React 18**
- **Vite**
- **TailwindCSS** e Tailwind Merge
- **React Router**
- **@tanstack/react-query**
- **Zod** (validação de dados)
- **Axios** (requisições HTTP)
- **Lucide React** (ícones)
- **Date-fns** (manipulação de datas)
- **Sonner** (notificações)
- **Prettier/ESLint** (padronização de código)

## Como rodar

Clone o repositório

- git clone https://github.com/<seu-usuario>/fintrack.git

- cd fintrack

Instale as dependências
npm install

Inicie o app em modo desenvolvimento

- npm run dev

Acesse [http://localhost:5173](http://localhost:5173) no navegador.
